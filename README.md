# Auth + RBAC Demo

## Backend

```bash
cd backend
npm install
npm run dev
```

Default seeded users:
- `admin@example.com` / `admin123` (ADMIN)
- `user@example.com` / `user123` (USER)

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend expects backend at `http://localhost:4000` by default.
